# fintech-product-analytics

## Overview
This project analyzes the end-to-end Fintech user journey from onboarding to retention, 
and segments users by credit behavior to propose personalized product strategies.

## Objectives
- Analyze product funnel conversion
- Measure retention, churn, and reactivation
- Segment users by financial & credit behavior
- Propose data-driven product & growth strategies

## Tools
- SQL & Excel: data extraction, transformation & KPI tables
- Python: EDA, cohort retention, clustering
- Power BI & Looker Studio: interactive dashboards & visualisations
